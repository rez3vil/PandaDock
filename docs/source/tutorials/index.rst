Getting Started
---------------

If you're new to PandaDock, start with the :doc:`basic_docking` tutorial, which covers the fundamental concepts and basic usage patterns.

Advanced Topics
---------------

Once you're comfortable with basic docking, explore:

* :doc:`ml_enhanced_docking` - Learn how to use PandaML algorithm for superior affinity prediction
* :doc:`metal_docking_tutorial` - Master metal docking for metalloproteins with coordination constraints
* :doc:`comprehensive_benchmark_tutorial` - Reproduce comprehensive benchmark results on 5,316 PDBbind complexes
* :doc:`virtual_screening` - Discover how to screen large compound libraries efficiently

Each tutorial includes:

* Step-by-step instructions
* Complete working code examples
* Troubleshooting tips
* Performance optimization guidance
* Best practices and recommendations

Prerequisites
-------------

Before starting the tutorials, ensure you have:

* PandaDock installed (see :doc:`../user_guide/installation`)
* Basic understanding of molecular docking concepts
* Python programming knowledge
* Sample data files (provided in each tutorial)

Getting Help
------------

If you encounter issues while following the tutorials:

* Check the :doc:`../user_guide/troubleshooting` guide
* Review the :doc:`../api/index` documentation
* Visit our GitHub repository for code examples
* Join the PandaDock community for support

Tutorial Structure
------------------

Each tutorial follows a consistent structure:

1. **Introduction** - Overview of concepts and goals
2. **Prerequisites** - Required knowledge and setup
3. **Step-by-step Instructions** - Detailed implementation guide
4. **Complete Examples** - Working code you can run
5. **Analysis and Interpretation** - Understanding results
6. **Best Practices** - Optimization and troubleshooting
7. **Next Steps** - References to related topics

Happy docking!
